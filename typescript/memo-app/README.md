# メモ帳アプリ

React + TypeScript で作成したシンプルなメモ帳アプリです。  
ローカルストレージを活用して、メモの保存・編集・削除ができます。

---

## 機能

- メモの新規作成 
- メモの編集 
- メモの削除 
- ローカルストレージに保存 
- 選択したメモの内容をフォームに自動反映 

---

## 技術スタック

- [React](https://reactjs.org/)
- [TypeScript](https://www.typescriptlang.org/)
- CSS（プレーンなスタイル）

---

## インストール方法

ローカルでアプリを実行するには、以下の手順に従ってください：

1. 依存関係をインストールします：
   ```
   npm install
   ```
2. アプリを起動します：
   ```
   npm start
   ```
3. ブラウザを開き、http://localhost:3000 にアクセスします。
